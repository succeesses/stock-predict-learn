  根因分析：
  - 原错误理解：认为 Qlib $open/$high/$low/$close 是原始不复权价格，需要 × 
  $factor 得到前复权

  是复权因子
  - 要得到不复权价格（和行情软件一致的真实价格），需要 不复权 = 前复权 / $factor

  预测。

  │        功能         │                       说明                        │
  ├─────────────────────┼───────────────────────────────────────────────────┤

  ├─────────────────────┼───────────────────────────────────────────────────┤
  │ 📊 支持多种数据格式 │ CSV/Feather 等金融数据格式                        │
  ├─────────────────────┼───────────────────────────────────────────────────┤

  ├─────────────────────┼───────────────────────────────────────────────────┤
  │ 🖥️ 多设备支持       │ CPU / NVIDIA CUDA / Apple Silicon MPS             │
  ├─────────────────────┼───────────────────────────────────────────────────┤
  │ 📈 K线图显示        │ 使用 Plotly.js 绘制专业金融K线图                  │
  ├─────────────────────┼───────────────────────────────────────────────────┤

  └─────────────────────┴───────────────────────────────────────────────────┘

  使用方法

  cd webui
  conda activate kronos
  pip install -r requirements.txt

  Windows 启动：

  .\start.ps1

  Linux/macOS 启动：

  chmod +x start.sh
  ./start.sh

  启动成功后，浏览器访问 http://localhost:7070 即可使用。

  使用步骤

  1. 加载数据 → 选择你的 CSV 数据文件
  2. 加载模型 → 选择模型大小 (mini/small/base) 和计算设备 (CPU/GPU)
  3. 设置参数 → 调节 Temperature / top_p / 采样数量
  4. 点击预测 → 等待生成结果
  5. 查看结果 → 网页上显示K线图和对比分析
